# Section 6: High-Entropy File Indicators
# ---------------------------------------------------------------------------
# NOTE: High-entropy detection cannot be done via simple path patterns.
# The scanner uses 'ent' or custom entropy calculation on sampled files.
# Files with Shannon entropy > 7.9 (out of 8.0) across the first 4096 bytes
# are flagged as potentially encrypted/compressed/packed.
# The following directories are priority targets for entropy scanning:
# ENTROPY_SCAN:C:\Users\*\Documents\
# ENTROPY_SCAN:C:\Users\*\Desktop\
# ENTROPY_SCAN:C:\Users\*\Pictures\
